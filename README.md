##### 有意思的小玩意:

- asciinema(cli): shell命令行录屏工具
- figlet (cli):	ascii艺术字生成工具
- translate-shell (cli):    命令行在线翻译工具
- memo:　Manjaro论坛上一个老哥写的下拉式便笺
- yakuake:    KDE自带的下拉式终端
- bat:  完美替代 cat
- tldr: 我用它替代了 info 命令
- rg: 听说是用 Rust 实现的，性能比 grep 要好
- fd: find 的完美替代，同为 Rust 实现
- termchat: 一个Rust实现的无服务器局域网终端聊天工具
- cloc: 代码行数统计工具（cli）
